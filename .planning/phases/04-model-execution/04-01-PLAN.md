### Task 3 — Implement _execute_chatterbox_tts()

**File:** `harmonyspeech/task_handler/model_runner_base.py`

**Input tuple from prepare:** `(text, conditionals_or_None, exaggeration, cfg_weight, temperature, repetition_penalty, top_p, min_p)`

**Output class:** `TextToSpeechRequestOutput`

**Implementation:**

```python
def _execute_chatterbox_tts(self, inputs, requests_to_batch):
    """Execute ChatterboxTTS to generate speech audio from text."""

    def synthesize(input_params):
        text, conditionals, exaggeration, cfg_weight, temperature, repetition_penalty, top_p, min_p = input_params

        if conditionals is not None:
            # Pre-computed Conditionals provided — move to device and assign
            self.model.conds = conditionals.to(self.device)
        else:
            # No pre-computed Conditionals — use raw audio bytes from request
            # Re-read from initial_request (prepare function does not include raw bytes in tuple)
            audio_bytes = base64.b64decode(initial_request.request_data.input_audio)
            audio_buf = io.BytesIO(audio_bytes)
            self.model.prepare_conditionals(audio_buf, exaggeration=exaggeration)

        wav = self.model.generate(
            text,
            exaggeration=exaggeration,
            cfg_weight=cfg_weight,
            temperature=temperature,
            repetition_penalty=repetition_penalty,
            top_p=top_p,
            min_p=min_p,
        )

        # Encode tensor to WAV base64
        audio_array = wav.squeeze(0).detach().cpu().numpy()
        with io.BytesIO() as wav_buffer:
            sf.write(wav_buffer, audio_array, self.model.sr, format='WAV')
            wav_bytes = wav_buffer.getvalue()
        return base64.b64encode(wav_bytes).decode('utf-8')

    outputs = []
    for i, x in enumerate(inputs):
        initial_request = requests_to_batch[i]
        inference_result = synthesize(x)
        result = self._build_result(initial_request, inference_result, TextToSpeechRequestOutput)
        outputs.append(result)
    return outputs
```

**Key design notes:**
- When `conditionals is None`, audio bytes come from `initial_request.request_data.input_audio` (base64-encoded string) — the prepare function intentionally omits raw bytes from the tuple (it only deserializes Conditionals objects)
- Use `conditionals.to(self.device)` to move pre-computed Conditionals to the correct device
- Assign to `self.model.conds` (single-threaded model runner — safe)
- Do NOT pass `audio_prompt_path` to `generate()` — leave as default `None`

**Checklist:**
- [ ] Unpacks all 8 tuple fields correctly
- [ ] Conditionals injection path: `self.model.conds = conditionals.to(self.device)`
- [ ] Audio bytes path: base64-decode `initial_request.request_data.input_audio`, wrap in `io.BytesIO`, call `prepare_conditionals()`
- [ ] Calls `self.model.generate()` with all 6 generation params
- [ ] Encodes result tensor via `wav.squeeze(0).detach().cpu().numpy()` + `sf.write` + `base64.b64encode`
- [ ] Returns `TextToSpeechRequestOutput` via `_build_result`

### Task 6 — Implement _execute_chatterbox_embedding()

**File:** `harmonyspeech/task_handler/model_runner_base.py`

**Input from prepare:** `audio_bytes` (single raw bytes value per request — NOT a tuple)

**Output class:** `SpeechEmbeddingRequestOutput`

**Implementation:**

```python
def _execute_chatterbox_embedding(self, inputs, requests_to_batch):
    """Execute ChatterboxEmbedding to compute voice Conditionals from audio."""

    def compute_embedding(audio_bytes):
        audio_buf = io.BytesIO(audio_bytes)
        self.model.prepare_conditionals(audio_buf)
        conditionals = self.model.conds

        # Serialize Conditionals to BytesIO using torch.save
        # (bypasses Conditionals.save(fpath) which requires a file path)
        buf = io.BytesIO()
        arg_dict = dict(t3=conditionals.t3.__dict__, gen=conditionals.gen)
        torch.save(arg_dict, buf)
        buf.seek(0)
        return base64.b64encode(buf.getvalue()).decode('utf-8')

    outputs = []
    for i, audio_bytes in enumerate(inputs):
        initial_request = requests_to_batch[i]
        inference_result = compute_embedding(audio_bytes)
        result = self._build_result(initial_request, inference_result, SpeechEmbeddingRequestOutput)
        outputs.append(result)
    return outputs
```

**Key design notes:**
- `inputs` is a list of raw bytes (one per request), NOT a list of tuples — the loop variable is `audio_bytes` directly
- `Conditionals.save(fpath)` only accepts a file path, but `torch.save()` (called internally) accepts file-like objects — bypass by building `arg_dict` and calling `torch.save(arg_dict, buf)` directly
- The `arg_dict` structure mirrors what `Conditionals.save()` would produce: `{"t3": conditionals.t3.__dict__, "gen": conditionals.gen}`

**Checklist:**
- [ ] Iterates inputs as raw bytes (no tuple unpacking needed)
- [ ] Wraps bytes in `io.BytesIO` for `prepare_conditionals()`
- [ ] Reads `self.model.conds` after prepare
- [ ] Serializes via `torch.save(dict(t3=..., gen=...), buf)`; seeks to 0 before encoding
- [ ] Returns `SpeechEmbeddingRequestOutput` via `_build_result`

### Task 7 — Implement _execute_chatterbox_vc()

**File:** `harmonyspeech/task_handler/model_runner_base.py`

**Input tuple from prepare:** `(source_bytes, target_conditionals_or_None, target_audio_bytes_or_None)`

**Output class:** `VoiceConversionRequestOutput`

**Implementation:**

```python
def _execute_chatterbox_vc(self, inputs, requests_to_batch):
    """Execute ChatterboxVC to perform voice conversion."""

    def convert(input_params):
        source_bytes, target_conditionals, target_audio_bytes = input_params

        if target_conditionals is not None:
            # Pre-computed Conditionals — extract .gen dict and assign to model.ref_dict
            self.model.ref_dict = target_conditionals.to(self.device).gen
        else:
            # Use target audio bytes to set target voice
            target_buf = io.BytesIO(target_audio_bytes)
            self.model.set_target_voice(target_buf)

        source_buf = io.BytesIO(source_bytes)
        wav = self.model.generate(audio=source_buf)

        # Encode tensor to WAV base64
        audio_array = wav.squeeze(0).detach().cpu().numpy()
        with io.BytesIO() as wav_buffer:
            sf.write(wav_buffer, audio_array, self.model.sr, format='WAV')
            wav_bytes = wav_buffer.getvalue()
        return base64.b64encode(wav_bytes).decode('utf-8')

    outputs = []
    for i, x in enumerate(inputs):
        initial_request = requests_to_batch[i]
        inference_result = convert(x)
        result = self._build_result(initial_request, inference_result, VoiceConversionRequestOutput)
        outputs.append(result)
    return outputs
```

**Key design notes:**
- When using pre-computed Conditionals for VC, the target voice is set via `self.model.ref_dict = target_conditionals.to(self.device).gen` (NOT `self.model.conds`)
- When using audio bytes, call `self.model.set_target_voice(io.BytesIO(target_audio_bytes))`
- Source audio is always passed as `audio=io.BytesIO(source_bytes)` to `model.generate()`
- Phase 3 guarantees exactly one of `target_conditionals` or `target_audio_bytes` is non-None (ValueError raised otherwise)

- [ ] Audio bytes path: `self.model.set_target_voice(io.BytesIO(target_audio_bytes))`
