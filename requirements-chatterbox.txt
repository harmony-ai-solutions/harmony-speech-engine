# Chatterbox TTS — separate install required due to conflicting pip metadata.

# transformers==4.46.3, safetensors==0.5.3, diffusers==0.29.0) that would
# downgrade HSE's core dependencies if installed normally. The code itself is
# compatible with newer versions; the pins are overcautious.

#
#   pip install --no-deps -r requirements-chatterbox.txt
#
# The --no-deps flag is required here — it tells pip to skip chatterbox-tts's
# declared dependency pins entirely. The safe transitive deps are already
# listed in requirements-common.txt and installed as part of the normal install.
#
# IMPORTANT: When upgrading this version pin, follow the checklist in
# .planning/codebase/CONCERNS.md § "Chatterbox TTS Dependency Pinning".
